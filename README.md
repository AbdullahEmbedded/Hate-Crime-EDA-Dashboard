# 📊 Hate Crime Dashboard – Interactive Data Analysis & Visualization

An interactive dashboard built with **Python** for exploring, cleaning, filtering, and visualizing hate crime data. The project demonstrates the complete data analysis workflow, from raw dataset preprocessing to an interactive web dashboard deployed on **Hugging Face Spaces**.

---

## 🚀 Live Demo

👉 **Try the dashboard online:**

**https://huggingface.co/spaces/hate-crime-explorer-dashboard/hate-crime-dashboard**

No installation is required. Open the link above and explore the dashboard directly in your browser.

---

## 📖 Project Overview

The goal of this project is to transform raw hate crime data into meaningful visual insights through an interactive dashboard.

The project includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Interactive filtering
- Statistical summaries
- Dynamic visualizations
- Web dashboard deployment

This project demonstrates practical skills in **Python programming**, **data analysis**, **data visualization**, and **web application deployment**.

# ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Hate-Crime-EDA-Dashboard.git
```

Open the project folder:

```bash
cd Hate-Crime-EDA-Dashboard
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
